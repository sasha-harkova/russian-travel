# Project: "Travel in Russia"

One-page website about the most beatiful places in Russia and interesting facts about them.
Has recommendation content:  **what you need to see in Russia**.

Here were used:
* grid layout;
* flexbox layout;
* adaptive layout;
* positioning properties;
* BEM technology;
* animations;
* etc
